# 🐍 Snake Game

A classic Snake Game built with HTML, CSS, and JavaScript — fully playable in the browser with no installations required.

## 🎮 How to Play

- Use the **Arrow Keys** to control the snake
- Eat the food to grow longer and increase your score
- Avoid hitting the **walls** or your own **tail**
- Game over when you collide!

## 🛠️ Built With

- HTML5 Canvas
- CSS3
- Vanilla JavaScript

## 🚀 Getting Started

1. Clone the repository:
```bash
   git clone https://github.com/Poojanshu/Snake-Game.git
```
2. Open `index.html` in your browser and start playing!
